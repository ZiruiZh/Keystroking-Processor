INSTALLATION INSTRUCTIONS FOR macOS SEQUOIA 15.6.1
==================================================

QUICK START:
1. Double-click "TypingAnalysisWorkstation.app"
2. If macOS blocks it, right-click and select "Open"
3. Click "Open" when prompted

DETAILED INSTALLATION:

Method 1 - Applications Folder (Recommended):
1. Copy "TypingAnalysisWorkstation.app" to your Applications folder
2. Open Applications folder in Finder
3. Find "TypingAnalysisWorkstation.app"
4. Double-click to launch

Method 2 - Desktop:
1. Keep "TypingAnalysisWorkstation.app" on your Desktop
2. Double-click to launch directly

SECURITY SETTINGS:
If macOS blocks the app:
1. Go to System Preferences > Security & Privacy
2. Click "Open Anyway" next to the blocked app
3. Or right-click the app and select "Open"

FIRST RUN:
- The app will create an output folder: ~/xml-to-text-video-output/
- Settings are automatically saved for future use

SYSTEM REQUIREMENTS:
- macOS Sequoia 15.6.1 or later
- Apple Silicon (M1/M2/M3) or Intel x64
- 4GB RAM minimum
- 500MB free disk space

TROUBLESHOOTING:
- App won't open: Right-click and select "Open"
- Permission denied: Check Security & Privacy settings
- No output folder: Check disk space and permissions
