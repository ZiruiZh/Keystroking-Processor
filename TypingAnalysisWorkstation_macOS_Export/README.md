# Typing Analysis Workstation for macOS

## Overview
Professional scientific tool for analyzing typing patterns and generating video visualizations from XML, IDFX, and text files.

## System Requirements
- macOS Sequoia 15.6.1 or later
- Apple Silicon (M1/M2/M3) or Intel x64 processor
- 4GB RAM minimum
- 500MB free disk space

## Installation
1. Download the `TypingAnalysisWorkstation.app` file
2. Move it to your Applications folder
3. Right-click the app and select "Open" (first time only)
4. Click "Open" when macOS asks for permission

## Features
- **Multi-format Support**: XML, IDFX, TXT, and DOCX files
- **Professional UI**: Scientific workstation interface with dark theme
- **Batch Processing**: Queue multiple files for processing
- **Video Generation**: Create MP4 videos from typing data
- **Moving Window**: Advanced masking and windowing features
- **Customizable Settings**: Font, speed, margins, and more

## Usage
1. **Launch the App**: Double-click `TypingAnalysisWorkstation.app`
2. **Select Files**: Use "Select File" or "Add to Queue" buttons
3. **Configure Settings**: Adjust text, window, and speed settings
4. **Generate Videos**: Click "Generate Video" or "Process Queue"

## File Types Supported
- **XML Files**: Typing data in XML format
- **IDFX Files**: Interactive Data Format XML files
- **TXT Files**: Plain text files
- **DOCX Files**: Microsoft Word documents

## Output
- Videos are saved to: `~/xml-to-text-video-output/`
- Settings are automatically saved and restored

## Troubleshooting
- If the app won't open, right-click and select "Open"
- For permission issues, go to System Preferences > Security & Privacy
- Ensure you have sufficient disk space for video output
